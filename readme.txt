This is a wordpress plugin for share post on persian and english social networks.

== Description ==

Stack Design  socail share plugin is easy way to create share button in below of your blog post.

Major features in Akismet include:

* Support some famous persian and english social network. 



== Installation ==

Upload the stack design social share  plugin to your blog, Activate it,in setting  social share menu check what social network you want to display. then enjoy it.
